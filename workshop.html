Tentang Workshop

Workshop Claude Code adalah pelatihan intensif satu hari yang dirancang untuk developer, data scientist, dan tech enthusiast yang ingin memanfaatkan kecerdasan buatan dalam workflow sehari-hari.

Kamu akan belajar langsung cara menggunakan Claude Code — mulai dari setup awal, menulis dan me-review kode bersama AI, hingga mengotomasi tugas-tugas yang memakan waktu.

Yang akan kamu pelajari:

  • Setup & konfigurasi Claude Code dari nol
  • Teknik prompt engineering untuk hasil kode terbaik
  • Debugging & refactoring dengan bantuan AI
  • Integrasi Claude Code ke project nyata (studi kasus)
  • Tips & trik produktivitas dari para expert
  • Sesi Q&A eksklusif dan networking session
